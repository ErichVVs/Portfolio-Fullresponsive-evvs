PORTFOLIO

Panadería de Xocoatl
Es una panadería ficticia especializada en bollería y horneado con chocolate y cacao, con la búsqueda de tener una forma de facilitar un catálogo web de productos y un botón directo hacía su WhatsApp y poder confirmar el pedido. Creado con HTML, CSS, SASS y Bootstrap. Manejando un controlador de versión mediante Git y un pages con Github. Para el diseño me inspiré en una línea minimalista web responsive en sitios como; la Esperanza y el Globo.

Los Velts
Es una banda de rock alternativo mexicana, formada por músicos de origen sinaloense. Sitio web estático para consolidar una presencia en la web y reunir sus redes de Facebook, Instagram y Spotify, además de dar información sobre las fechas en que se estarán presentando. Creado con HTML, CSS, SASS y Bootstrap. Manejando un controlador de versión mediante Git y un pages con Github. Para el diseño me inspiré en sitios como; Avril Lavinge y Blink 182.
